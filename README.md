# Clinvedica

Clinvedica is a web application built with [Next.js](https://nextjs.org) using the App Router. It is optimized for SEO, fast page loads, and modern developer experience.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: (add your styling solution, e.g. Tailwind CSS / CSS Modules / etc.)
- **Package manager**: npm / yarn / pnpm / bun (pick one and update this)
- **Deployment**: Vercel (recommended) or any Node.js hosting

## Getting Started

### Prerequisites

- **Node.js**: v18+ (recommended: LTS)
- **Package manager**: npm, yarn, pnpm, or bun

### Installation

# Clone the repo
git clone <your-repo-url>
cd clinvedica

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
# or
bun install
